크롤링 설정

검색은 대소문자 구분 없이 처리되며, 날짜는 YYYY.MM.DD / YYYY-MM-DD / YYYY/MM/DD 형식을 지원합니다.
- Google 화면 구조가 바뀌면 headful 모드에서 확인하면서 다시 실행하는 편이 안전합니다.
- 리뷰 날짜가 6 days ago / 1 week ago / 3달 전 / 7주 전처럼 보이면, 실행일 기준으로 실제 날짜를 계산해 저장합니다.
- 동일 상품이 여러 쇼핑 카드로 보이더라도, 리뷰 수가 동일한 카드 중 1개만 선택해서 수집하도록 설계했습니다.
- 상품명은 콤마(,) 또는 공백을 포함한 슬래시( / )로 여러 개 입력할 수 있으며, 입력 순서대로 연속 수집합니다.
- 우측 패널의 사용자 리뷰 → 관련성순 → 최신순 → 리뷰 더보기 흐름에서, 최신순 상태가 실제로 확인되지 않으면 리뷰 추출을 시작하지 않도록 유지했습니다.
- 각 리뷰의 본문 더보기 / Read more / 더보기도 눌러서, 가능한 경우 리뷰 전문을 CSV에 넣도록 보강했습니다.

계정: Dr1brx / 별점: 5 / 날짜: 3주 전 → 실행일 기준 실제 날짜
